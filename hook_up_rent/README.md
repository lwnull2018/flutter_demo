# hook_up_rent

好客租房项目实战

解析JSON需要的依赖：
dependencies:
    flutter:
        #json序列化工具
        json_annotation: ^4.9.0
        build_runner: ^2.4.9

dev_dependencies:
    build_runner: ^2.3.3
    json_serializable: ^6.8.0

执行命令：
    dart run build_runner build ， 生成解析JSON的代码
